比例（P）控制规律：

  • 稳态误差：不同，系统余差不同，。规律：随着增大（$$减小）而减小。有差调节。
  • 动态过程：随着增大（$$减小），系统响应速度加快，系统超调量增加，调节时间延长。
  • 增大到一定值（某一临界值）后，闭环系统变得不稳定。

积分控制的特点

PI控制的特点

为比例带，可视情况取正值或负值；为积分时间，愈小，积分部分所占比重愈大。

比例积分对控制过程的影响

总结（重点）

  • 稳态性能：无差调节，系统稳定于给定值。
  • 动态性能：随着积分时间的减少，积分控制作用增强，上升时间变快，闭环系统的稳定性变差。
  • 与纯P调节相比，PI调节的过渡过程更长。
  • PI控制既保证系统的稳定性，又能消除余差。适用于对于过渡过程要求不高，要求无差的系统。

比例微分控制规律

比例微分输出的大小与偏差变化速度及微分时间成正比。微分时间越大，微分作用越强。

PD控制的特点

  • PD调节也是有差调节，与P调节相同。
  • 适度引入微分动作可以提高控制系统的稳定性。此时可以少许减少比例带，以保持衰减率不变。

积分分离法

当控制偏差较大时，如大于人为设定的某阈值时，取消积分作用，以减小超调；而当控制偏差较小时，再引入积分控制，以消除余差，提高控制精度。

遇限削弱积分法

数字PID算法中，计算控制变量u(k)时，先对上一时刻的控制变量u(k-1)进行判断，若u(k-1)>umax，则只累加负偏差，若u(k-1)<umin，则只累加正偏差。这样可以避免长时间停留在饱和区。

简单控制系统的组成

单回路控制系统的方框图：

是指一个测量变送器、一个调节器、一个执行器连同被控过程组成的、对一个被控参数进行控制的单回路负反馈控制系统。

被调量的选择

一般原则：

  1. 选择对产品的产量和质量、安全生产、经济运行和环境保护具有决定作用的、可直接测量的工艺参数为被控量。
  2. 当不能用直接参数作为被控量时，应选择一个与直接参数有单值函数关系的间接参数作为被控参数。
  3. 被控参数必须具有足够大的灵敏度
  4. 被控参数的选取必须考虑工艺过程的合理性和所用仪表的性能及成本。

控制通道特性和扰动通道特性对调节质量的影响—＞调节变量的选择

根据过程特性选择控制参数的一般原则：

  • 控制通道的放大系数要适当选大一些；时间常数要适当小一些；纯滞后时间越小越好，在有纯延时的情况下，$$与T之比应小于1.
  • 扰动通道的放大系数应尽可能小；时间常数要大；扰动引入系统的位置要远离控制过程（即靠近调节阀）；容量滞后愈大，愈有利于控制。
  • 广义过程（包括调节阀和测量变送器）由几个一阶环节组成，在选择控制参数时，应尽量将几个时间常数数值错开，使其中一个时间常数比其他时间常数大得多，同时注意减小其他时间常数。
  • 注意工艺操作的合理性、经济型。应是工艺上允许加以控制的可控变量。一般不应该选择工艺上的主要物料（产品）作为控制参数。

干扰通道的特征参数对控制质量的影响

  1. 越大，扰动作用下系统的动态偏差、稳态误差（静态偏差）越大。
  2. 增加，被调量y对扰动的响应速度变慢，动态偏差减少。
  3. 扰动通道迟延时间的存在仅使被调量在时间轴上平移了一个$$时间，并不影响系统的控制质量。
  4. 当有多个扰动时，扰动进入系统的位置离输出（被调量）越远，对系统控制质量影响就越小。

串级控制系统

串级控制系统的方框图

串级控制系统性能上的主要优势

  • 改善了对象的特性，提高系统的工作频率，加快了调节过程。
  • 串级控制系统具有较强的抗干扰能力（尤其是二次扰动）
  • 副回路的存在，使串级系统有一定的自适应能力（鲁棒性）

串级系统主副回路的设计

  1. 使系统中主要干扰包含在副环内。
  2. 用于克服对象的容量滞后：应选择一个滞后较小的辅助变量组成副回路。副环的时间常数不能过大，以防共振；也不能过小，力求多包含一些干扰。
  3. 应使非线性环节尽量包含在副环中。
  4. 当对象具有较大的纯滞后时，应使副环尽量少包含或不包含纯滞后。
  5. 考虑主、副对象时间常数的匹配，防止共振发生。

前馈控制的优势与局限

  1. 前馈控制直接按干扰作用的大小进行控制，比反馈控制要及时。
  2. 前馈控制属于开环控制，控制效果不能通过反馈验证，因此对控制器设计的要求比较严格。
  3. 前馈控制系统只能用来克服生产过程中主要的、可测不可控的扰动，而不能克服不可测的扰动。
  4. 前馈控制系统一般只能实现局部补偿而不能保证被调量的完全不变，一种前馈控制只能控制一种干扰。
  5. 前馈控制其是视对象特性而定的控制器，前馈控制算法依对象不同而不同。

前馈控制的选用规则

  • 若控制系统中控制通道的惯性和迟延较大，反馈控制达不到良好的控制效果时，可引入前馈控制。
  • 当系统中存在着经常变动、可测而不可控的扰动时，反馈控制难以克服扰动对被控量的影响，这时可引入前馈控制加以改善控制品质。
  • 当工艺上要求实现变量间的某种特殊的关系，而需要通过建立数学模型来实现控制时，可引入前馈控制。
  • 经济实用原则。在决定选用前馈控制方案后，当静态前馈能满足工艺要求时，就不必选用动态前馈。

相对增益的定义：

对于某通道，定义该通道在其他回路均为开环时的放大系数为第一放大系数，定义该通道在其他回路均为闭环时的放大系数为第二放大系数，两者之比为，称为相对增益。

相对增益是一种尺度，用来衡量在耦合情况下，相对于过程中其他调节量对被控量的影响程度。

选择控制回路的原则：

  1. 一个被控量应选择最大且接近1的正相对增益的控制量与之匹配；
  2. 不能用相对增益为负数的被控量和控制量构成控制回路
  3. 相对增益为方阵意味着控制量和被控量数目相同；如果被控量和控制量数目不同，如两个被控量和三个控制量，则有三对可能的控制量组合，就可以得到三种不同的相对增益矩阵，选择控制回路时都应进行考察。
  4. 相对增益矩阵从稳态上衡量变量之间的关联程度，所以据此选择控制回路不能保证动态关联最小。
