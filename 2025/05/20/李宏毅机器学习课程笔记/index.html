A Case Study on the Prediction of no. of views of a YouTube Channel

  1. Function with Unknow Parameters: ——Model

  2. Define Loss from Training Data: ——损失函数

    Loss is a function of parameters, it represents how good a set of values is.

    Loss: ——自行定义，可正可负

    —— is mean absolute error(MAE, 平均绝对误差)

    —— is mean square error(MSE, 均方误差)

    if and are both probability distributions——Cross-entropy(交叉熵)

  3. Optimization: $w^{},b^{}=arg \min_{w,b} L $

    Gradient Descent(梯度下降)：

    • (Randomly)Pick an initial value ,
    • Compute $\frac{\partial L}{\partial w} |{w=w^0,b=b^0}\frac{\partial L}{\partial b} |{w=w^0,b=b^0}$
    • Update and iteratively(迭代)
